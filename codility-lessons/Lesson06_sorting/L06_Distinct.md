# Codility - L06. Distinct

## Description
	
	배열 A를 받아서 unique한 원소의 개수를 리턴하는 문제.
	<br>
	예를 들어, A = [2, 1, 1, 2, 3, 1] 일때 return은 3이다.
	<br>
	배열에 1, 2, 3이 unique하게 존재하기때문이다.
