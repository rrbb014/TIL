# git initialization in CLI

CLI에서 더 빠르게 repo를 생성하자!

1. `mkdir "DIR_NAME"`
2. `cd "DIR_NAME"`&nbsp
3. `git init`
4. `touch README.md`
5. `git add README.md`
6. `git commit -m "first commit"`
7. `git remote add origin https://github.com/"ID"/"REPO_NAME".git`
8. `git push --set-upstream origin master`
